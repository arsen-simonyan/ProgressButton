### Integration

The lib is available on Maven Central, you can find it with [Gradle, please]

```
allprojects {
   repositories {
      ...
      maven { url 'https://jitpack.io' }
   }
}
```

```
dependencies {
    implementation 'https://github.com/arsen-simonyan/ProgressButton:$latest'
}
```
